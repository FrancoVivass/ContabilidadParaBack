📌 CÓMO ABRIR EL SITIO WEB
════════════════════════════════════════════════════════════════

OPCIÓN 1: Servidor Local con Python (RECOMENDADO)
--------------------------------------------------
1. Abre una terminal/PowerShell en esta carpeta
2. Ejecuta: python -m http.server 8000
3. Abre tu navegador en: http://localhost:8000
4. ¡Listo! El sitio debería funcionar perfectamente

OPCIÓN 2: Abrir directamente el archivo
----------------------------------------
1. Doble clic en "index.html"
2. Se abrirá en tu navegador predeterminado

⚠️ NOTA: Si el header y footer no cargan, usa la Opción 1

🛠️ SI ALGO NO FUNCIONA
════════════════════════════════════════════════════════════════

Problema: Header/Footer no aparecen
Solución: Usa un servidor local (Python, Node, PHP)

Problema: El clima no se muestra
Solución: Permite acceso a la ubicación en tu navegador

Problema: Imágenes no cargan
Solución: Verifica que la carpeta "assets" esté completa
